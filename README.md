# prework-study-guide
## Description

- This is a website that is meant to help bootcamp students study the basics of coding.
- This was made in an effort to practice and use the topics listed.
- Now there is a website that outlines the basic tenets of what skills this bootcamp will build upon.
- I learned how to use HTML, CSS, Git/Github, and JavaScript.

## Usage

To use this Prework Study Guide, you can review the notes in each section. Each section contains notes about coding topic, like HTML, CSS, JavaScript, Git. For suggestions on what to study first, open the Chrome DevTools by pressing Command+Option+I (macOS) or Control+Shift+I (Windows). A console panel should open either below or to the side of the webpage in the browser. There you will see a list of topics we learned from the prework along with a suggestion on which topic to study first.
